## 🎉 Overview
**EventHub** is a simple, accessible, and SEO‑optimized event management platform built as part of RRWD Prep Week. It allows users to browse events, create new events, manage tickets, and navigate through a clean, single‑page interface.

The project focuses on:
- Semantic HTML  
- Accessibility (WCAG‑aligned)  
- SEO best practices  
- Clean navigation and UX  
- GitHub Pages deployment  

---

## 🚀 Features

### 🔹 Event Browsing
Users can view upcoming events with images, descriptions, dates, and locations.

### 🔹 Event Creation
A simple form allows users to create new events with:
- Event name  
- Date & time  
- Location  

### 🔹 Ticket Management
Users can view events they’ve registered for.

### 🔹 Profile Section
Displays user information and links to event history.

### 🔹 Accessibility Enhancements
- Skip‑to‑content link  
- ARIA attributes  
- Keyboard‑friendly navigation  
- Screen‑reader live region  

### 🔹 SEO Enhancements
- Keyword‑optimized headings  
- Optimized image filenames  
- Descriptive alt text  
- Meta description  
- Organized heading structure

## ♿ Accessibility Notes
EventHub includes several accessibility improvements:

- ARIA labels and live regions  
- Keyboard‑accessible navigation  
- Proper form labels  
- Alt text for all images

These improvements help ensure the site is usable for a wide range of users.
